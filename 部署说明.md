# 📱 库存盘点 APP 部署说明

## 文件结构
```
pwa-app/
├── index.html        ← 主程序
├── manifest.json     ← APP 配置（名称、图标等）
├── sw.js             ← 离线缓存服务
├── icons/
│   ├── icon-192.png  ← APP 图标（小）
│   └── icon-512.png  ← APP 图标（大）
└── 部署说明.md
```

---

## 🚀 第一步：上传到 GitHub Pages（免费）

### 1. 注册 GitHub 账号
前往 https://github.com 注册（免费）

### 2. 新建仓库
- 点击右上角 `+` → `New repository`
- 仓库名填：`pandian`（或任意名称）
- 选 `Public`（公开）
- 点击 `Create repository`

### 3. 上传所有文件
- 点击 `uploading an existing file`
- 将整个文件夹的所有文件拖进去（注意 icons 文件夹也要上传）
- 点击 `Commit changes`

### 4. 开启 GitHub Pages
- 进入仓库 → `Settings` → `Pages`
- `Source` 选 `Deploy from a branch`
- `Branch` 选 `main`，文件夹选 `/ (root)`
- 点击 `Save`

### 5. 获取你的 APP 链接
等待约 1 分钟，你的 APP 地址为：
```
https://你的用户名.github.io/pandian/
```

## 📲 第二步：分享给他人安装

将上方链接发给需要使用的人，对方操作：

### Android 手机（Chrome 浏览器）
1. 用 Chrome 打开链接
2. 等页面加载完成
3. 点击右上角 `⋮` → `添加到主屏幕`
4. 点击 `安装` ✅

### iPhone（Safari 浏览器）
1. 用 Safari 打开链接
2. 点击底部 `分享` 按钮（方块+箭头图标）
3. 点击 `添加到主屏幕`
4. 点击右上角 `添加` ✅

## ✅ 安装后的效果
- 手机桌面出现「库存盘点」图标
- 点开后全屏显示，无浏览器地址栏

## ❓ 常见问题

**Q：需要付费吗？**
A：完全免费，GitHub Pages 不限流量。

**Q：数据安全吗？**
A：所有数据存在用户自己手机里（localStorage），不经过任何服务器。

**Q：更新程序怎么办？**
A：重新上传新的 index.html 到 GitHub，用户打开 APP 联网一次即可更新。
